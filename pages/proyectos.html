PROYECTOS

Hi-Chat

Aplicación de mensajería desarrollada en C++ con interfaz gráfica WinBGIm. Permite enviar y recibir mensajes en tiempo real mediante comunicación serial RS-232 (COM) entre dos equipos. Incluye personalización visual, soporte para emojis y guardado del historial del chat. Proyecto orientado a comprender comunicación de bajo nivel y manejo de interfaces.

Ver en GitHub

AquaTruck

Sistema web integral para empresas de reparto de soda. Gestiona clientes, productos, pedidos y repartos con una interfaz intuitiva y totalmente responsive. Desarrollado con Node.js + Express, MySQL, autenticación JWT y un frontend dinámico basado en modales y peticiones asincrónicas. Optimizado para consultas en tiempo real y administración completa desde el navegador.

Ver en GitHub

eCommerce

Aplicación web de catálogo de productos creada con HTML, CSS y JavaScript puro. Permite buscar por descripción, filtrar por rubro y alternar entre vista de lista y cuadrícula. Se conecta mediante fetch() a una API que obtiene datos desde MySQL. Proyecto enfocado en interactividad, consumo de APIs, responsive design y organización del DOM.

Ver en GitHub

Mi Portfolio Web

Sitio web personal diseñado para presentar mis habilidades, proyectos y experiencia de manera clara y profesional. Desarrollado con HTML, CSS y JavaScript, totalmente responsive y optimizado para ofrecer una navegación fluida. Funciona como carta de presentación online, permitiendo acceder a mis trabajos y repositorios.
